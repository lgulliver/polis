# Roadmap

## Milestone 1: infrastructure and one bot

- Initialize TypeScript workspace
- Connect one Mineflayer bot to a private PaperMC server
- Log structured events and periodic perception
- Support a few human-triggered deterministic commands

## Milestone 2: multiple bots

- Run multiple named bots concurrently
- Improve per-agent configuration
- Add basic experiment presets and operational scripts

## Milestone 3: survival skills

- Add simple deterministic survival behaviors
- Introduce navigation, regrouping, sheltering, and inventory management primitives
- Keep all low-level skills testable and schema-validated

## Milestone 4: commitments, trust, and shared projects

- Represent promises, obligations, and task ownership
- Track reliability and trust histories
- Enable collaborative projects such as farms, walls, or roads

## Milestone 5: LLM decision layer

- Add model-driven planning behind strict JSON schemas
- Enforce an allowlisted action registry
- Keep the model separate from execution, logging, and safety checks

## Milestone 6: economy, governance, religion, conflict, dialect, scientists

- Economy: exchange, scarcity, and division of labor
- Governance: norms, rules, offices, and enforcement
- Religion and ritual: symbolic acts, myths, taboos, seasonal practices
- Conflict: factions, feuds, deterrence, peacemaking
- Dialect: local vocabularies, abbreviations, and style drift
- Scientists: observation protocols, experiments, and knowledge accumulation
